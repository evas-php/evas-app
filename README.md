# evas-app
Evas PHP starter pack

## Install
1. Clone repository
```
git clone https://github.com/evas-php/evas-app myapp
```
2. Install dependencies
```
composer install
```

## Testing
1. Setup codeception
```
php vendor/bin/codecept bootstrap
```
2. Run tests
```
php vendor/bin/codecept run unit vendor/evas-php/
```
